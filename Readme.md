# 💰 FinAgentX

## Where AI Meets Investing

FinAgentX is an AI-powered financial assistant built using Streamlit and Google Gemini AI. It helps users make informed financial decisions through AI-powered insights, portfolio analysis, budgeting tools, and financial calculators.

## Features

- 📊 Financial Dashboard
- 📈 AI Stock Analyzer
- 🤖 AI Financial Advisor
- 💼 Portfolio Analyzer
- 🧮 Financial Calculators
  - EMI Calculator
  - SIP Calculator
  - SWP Calculator
  - Lumpsum Calculator
  - CAGR Calculator
  - Retirement Calculator
  - Emergency Fund Calculator
  - Net Worth Calculator
  - Savings Rate Calculator
  - Financial Health Score
- 💰 Budget Planner
- ❤️ Financial Health Assessment

## Technology Stack

- Python
- Streamlit
- Google Gemini AI
- Pandas
- Plotly
- Python-dotenv

## Installation

Clone the repository:

```bash
git clone https://github.com/YashwanthKrishnaD/FinAgentX.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application:

```bash
streamlit run app.py
```
